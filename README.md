## Next.js + LangChain.js + DeepSeek 实现的一个 AI ChatBot

### 项目简介

本项目是一个基于 Next.js 和 LangChain.js 的 AI ChatBot 应用，使用 DeepSeek 作为底层模型。

### 技术栈

- Next.js
- LangChain.js
- DeepSeek

### 快速开始

```bash
pnpm install
```

在本地的 `.env.example` 改为 `.env.local`，并填入 DeepSeek 的 API key。

```bash
pnpm dev
```

在浏览器访问 `localhost:3000/chat`
